# archive/

This directory holds historical governance artifacts that have been
superseded by newer sources of truth. It exists so that prior
decisions remain inspectable without polluting the active surface of
the repository.

## Read-only invariant

Files under `archive/` are **read-only**. They are preserved as
historical record, not maintained as living documents.

- Do not edit files in `archive/`.
- Do not cite `archive/` as a source of current policy. For current
  policy, consult the active charter at
  `PROGRAM_CONTROL/DIRECTIVES/QUEUE/RRR-GOV-002.md`.
- Do not delete files in `archive/`. If an archived file must be
  corrected for factual reasons, a new charter amendment is required
  (see §9 of the active charter).
- New material does not belong here. New governance lives in
  `PROGRAM_CONTROL/DIRECTIVES/` under the active charter.
