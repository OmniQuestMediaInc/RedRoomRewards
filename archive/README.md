# archive/ — Read-Only Historical Record

## Invariant

This directory is **read-only**. Files here are archived copies of superseded
documents. They must not be edited, moved back to active locations, or deleted.

## Contents

| Path | Superseded by | Archived |
|------|--------------|---------|
| `governance/CLAUDE_2026-04-21.md` | `PROGRAM_CONTROL/DIRECTIVES/QUEUE/RRR-GOV-002.md` | 2026-04-21 |

## Why files land here

Documents move to `archive/` only when:

1. A CEO-approved directive explicitly supersedes them, **or**
2. A charter amendment marks them retired with a dated rationale.

## Who may modify this directory

No one. Read only. If a document needs to be un-archived, that requires an
explicit CEO directive — not an agent action.
